# 🚀 API Data Pipeline with Scheduler

## 📌 Overview
This project demonstrates an automated data pipeline that fetches data from an API, processes it, and stores it in a database.

---

## ⚙️ Tech Stack
- Python
- Pandas
- SQLite
- Requests
- Scheduler

---

## 🔄 Workflow
1. Extract data from API
2. Transform data
3. Load into database
4. Run automatically using scheduler

---

## ▶️ How to Run
pip install -r requirements.txt  
python scheduler.py  

---

## 📊 Output
Data stored in SQLite database (data.db)
